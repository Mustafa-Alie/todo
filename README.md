# Todo App – My First Full Stack Project

![App Preview](./preview.jpg)

Hello! 
 This is my **first full stack app**. I built it using **Vite, React,  TypeScript, Tailwind** for Frontend.
 **Node, Express, Prisma ORM, PostgreSQL**  for the Backend  

I kept the code **low-level, manual, and simple on purpose**. My goal was to **learn and practice** the core concepts of frontend and backend development.  

---

## 🛠️ Tech Stack

### Backend
- [Node.js](https://nodejs.org/) – JavaScript runtime  
- [Express](https://expressjs.com/) – Server framework  
- [TypeScript](https://www.typescriptlang.org/) – Typed JavaScript  
- [Prisma ORM](https://www.prisma.io/) – Database ORM  
- [PostgreSQL](https://www.postgresql.org/) – Database  
- [Neon](https://neon.tech/) – Serverless PostgreSQL hosting  
- [dotenv](https://www.npmjs.com/package/dotenv) – Environment variables  
- [argon2](https://www.npmjs.com/package/argon2) – Password hashing  
- [JSON Web Tokens (JWT)](https://jwt.io/) – Authentication  
- [cookie-parser](https://www.npmjs.com/package/cookie-parser) – Cookie handling  
- [CORS](https://www.npmjs.com/package/cors) – Cross-origin requests  
- [Nodemailer](https://nodemailer.com/) – Sending emails  
- [MailerSend](https://www.mailersend.com/) – Email service  
- [Google Cloud Platform](https://cloud.google.com/) – Hosting & services  
- [google-auth-library](https://www.npmjs.com/package/google-auth-library) – Google OAuth  

### Frontend
- [Vite](https://vitejs.dev/) – Build tool  
- [React](https://reactjs.org/) – UI library  
- [Tailwind CSS](https://tailwindcss.com/) – Styling  

---

## 🚀 Features

- User authentication with JWT and cookies  
- Create, read, update, and delete todos (CRUD)  
- Mark todos as completed  
- Email notifications using Nodemailer & MailerSend  
- OAuth login with Google  
- Fully typed with TypeScript
